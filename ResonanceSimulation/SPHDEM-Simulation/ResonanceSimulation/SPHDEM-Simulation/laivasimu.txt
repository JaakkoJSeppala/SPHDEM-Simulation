1. Projektin vaiheistus (kehitys pienissä paloissa)
Vaihe	Tavoite	Työkalut	Lopputulos
1. Perusrakenne	Luo WPF/Avalonia-sovellus, jossa on pääikkuna, simulaatiopaneeli ja asetuspaneeli.	Visual Studio / Rider, .NET 8	GUI + "Start Simulation" -painike
2. Partikkelijärjestelmä	Toteuta Particle-luokka ja listan hallinta.	C# (CPU)	Partikkelien liike (ilman fysiikkaa)
3. SPH-laskenta	Laske tiheys, paine ja painevoimat (vain neste).	CPU tai ComputeSharp GPU	Yksinkertainen SPH-neste
4. DEM-laskenta	Lisää kiinteät partikkelit ja törmäykset.	CPU	DEM-testisimulaatio
5. SPH–DEM-yhdistäminen	Toteuta neste–kiinteä rajapintavoimat.	CPU → GPU mahdollinen	Neste virtaa kiinteiden ympärillä
6. Visualisointi	Piirrä partikkelit WPF:llä tai OpenGL:llä.	SharpGL, SkiaSharp, HelixToolkit	2D/3D-näkymä
7. Käyttöliittymä ja asetukset	Lisää ohjaimet (aikaväli, hiukkasluku, viskositeetti).	WPF + MVVM	Käytettävä ohjelma
8. Validointi ja mittaukset	Vertaile tuloksia kirjallisuuteen (esim. hydrostaattinen tasapaino).	Tieteelliset lähteet	Validointi gradua varten
⚙️ 2. Suositeltu teknologia-alusta
Osa	Työkalu / kirjasto	Perustelu
Käyttöliittymä	WPF (.NET 8) tai AvaloniaUI	Helppo Windowsille, selkeä rakenne
3D/2D-visualisointi	HelixToolkit.WPF (3D) tai SkiaSharp (2D)	Tukee animaatiota ja zoomausta
Laskenta	C# + Parallel.For + Vector<float>	Erinomainen CPU-suorituskyky
GPU-laskenta (jos haluat laajentaa)	ComputeSharp	C#:llä suoraan HLSL-komentosarjoihin
Testaus	xUnit / NUnit	Jokainen laskentamoduuli voidaan testata
Dokumentointi	XML-kommentit + docfx	Automaattinen API-dokumentaatio graduun
